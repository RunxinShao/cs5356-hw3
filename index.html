Learning About the <details> Element

Click here to learn more about <details>

I discovered that the <details> element allows you to create a collapsible section on your webpage. It is useful for hiding lengthy or additional content until the user chooses to expand it. This helps in keeping the page layout clean and focused. I appreciate its simplicity and built-in functionality, which avoids the need for extra JavaScript.

Understanding the CSS filter Property

In this project, I used the filter property to apply a grayscale effect to images. The filter property enables a range of visual effects such as blurring, brightness adjustment, and contrast changes. I learned that combining it with CSS transitions can create smooth and visually appealing effects, enhancing user interaction.
